Python Seminar

The Python seminar met twice a week for 40 minutes. The students used chromebooks and an online site ( REPL.IT) to run Python. In addition we used a github repo to hold the code. For course material we drew on a book, Teach your Kids to Code, and copies were made available to the kids. The exercises were also posted at our own online site (ie this site) which the kids were encouraged to access on their own time.

In the first weeks we introduced basic ideas in programming (variables, assignment, if/else, loops) using Turtle graphics. The kids programmed the turtle to move around, making drawings with a virtual pen. They learned how to make use of repetition and to put together small pieces of code to achieve bigger goals like drawing a complicated figure. We also programmed calculations of statistics for the US women's soccer team. Some of the kids were studying pie charts in math class so we helped them write programs to draw colorful pie charts.

In the last weeks we investigated how ciphers, like Ceaser's cipher, can be used to send secret messages. In the process they used math concepts like remainder and invertable functions, as well as programming concepts like converting between numbers and characters. They really like some of the crazy symbols available in the Unicode character set.

Our goal was for the kids to get their feet wet, so to speak, and play around with some basic ideas in programming. We encouraged them to think about polishing their best work and including it in their portfolio for high school.
